# 📊 Power BI Practice Journey

This repository documents my step-by-step learning and practice in Power BI, starting from basic data cleaning to building interactive dashboards.

## Objective
To develop strong data analysis and visualization skills using Power BI through structured practice.

## 📂 Practice Levels

### 🟢 Level 1: Data Cleaning (Power Query)
- Removed duplicate rows
- Filtered data
- Changed data types

### 🟡 Level 2: Visualizations
- Bar chart (Top countries by population)
- Pie chart (Region distribution)
- Map visualization

### 🟠 Level 3: Data Analysis
- KPI cards (Total population, Avg GDP)
- Income group comparison

### 🔵 Level 4: Intermediate
- Created calculated column (Population Category)
- Added slicers
- Built scatter plot (GDP vs Population)

### 🔴 Level 5: Dashboard
- Combined all visuals into a single interactive dashboard
